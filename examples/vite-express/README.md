# Thanks for checking out Marko

This example showcases Marko with [Express](https://expressjs.com/) and [Vite](https://vite.dev).

# Installation

```
npm init marko -- --template vite-express
cd marko-app
npm install
npm run dev
```

## Overview

- Run `npm run dev` to start the development server
- Run `npm run build` to build a production-ready node.js server (`npm run start` to start it)
- Run `npm run preview` to run/preview the production server
